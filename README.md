# Lista-enlazada-simple

## Descripción
Implementación de una lista simplemente enlazada con 4 retos resueltos.

## Retos Implementados
1. countOccurrences: Cuenta cuántas veces aparece un valor en la lista.
2. clean: Elimina todos los nodos de la lista y retorna la cantidad eliminada.
3. reverseInPlace: Invierte el orden de la lista sin crear una nueva estructura.
4. removeDuplicates: Elimina valores duplicados conservando sola la primera aparicion.
